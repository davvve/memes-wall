# memes-wall

## Setup

1. Install dependencies

    `npm install`

2. Run backend server

    `node server.js`

3. Run frontend server

    `live-server --port=8080`



## Using MEMES-WALL

### Meme-Generator

`http://127.0.0.1:8080/social-wall.html?room=dave`
